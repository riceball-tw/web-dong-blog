## 前言

身為前端工程師或多或少都有被問過這樣的問題：「瀏覽器搜尋網址後發生了什麼事？」，當時把一些含糊的想法告訴對方，但感覺自己對於這方面知識還不夠深入，因此來補齊一下究竟發生了什麼事，記得當時回答是這樣：

1. 到域名伺服器找到網址對應的 IP 地址
2. 連結該 IP 地址，通常透過 HTTP 協定來回確定建立聯繫
3. 開始傳輸資料

這次我可以怎麼描述得更好呢？

## 第一步：確認問題邊界

1. 著重在瀏覽器的部分？
2. 著重在伺服器的部分？
3. 著重在網路協議的部分？
4. 整體流程？
