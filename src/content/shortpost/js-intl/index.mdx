近期重寫專案處理價錢標示需要替金額標示幣別、千分位分隔、小數點省略這些東西……舊專案做法是撰寫個別功能函式並組合起來去轉換金錢，使用起來蠻麻煩的。

於是翻了一下使用的 Vue i18n 多語系文件發現有支援數字轉換，而底層就是用到 JavaScript 原生的 [Intl](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl)。Intl 物件是 ECMAScript 國際化 API 的一個命名空間，它提供精確的字串對比、數字格式化與日期時間格式化。

![Intl 範例](./intl-example.webp)
