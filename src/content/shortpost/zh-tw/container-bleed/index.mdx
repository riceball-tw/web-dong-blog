近乎每個網頁為了確保內容保持在合理的範圍內，很多時候會需要在外層使用固定的最大尺寸來限制網頁內容。但近期我碰上需要溢出容器到頁面邊緣的視覺，這看起來就不是簡單設個最大範圍能達成的吧 ！☹

好險現代 CSS 具備 Grid、minmax、變數……等語法協助達成這類布局。[新的文章](https://www.webdong.dev/post/how-to-escape-container-by-using-css-grid/)帶你了解 J 個酷酷的 Grid 是如何做到的 👇🏻

```css
grid-template-columns: {
minmax(var(--container-padding-inline), 1fr)
minmax(0, var(--max-container-width-half))
minmax(0, var(--max-container-width-half))
minmax(var(--container-padding-inline), 1fr);
}
```
