一開始得知存在這個屬性還蠻興奮的，想像很多可以使用上的介面案例，甚至投影片功能也能輕鬆用原生卷軸來解決？不過實際用下去才會知道它們是截然不同的體驗。

這項屬性專注在卷軸的黏性吸附，因此無論是滾動卷軸還是觸控時都能生效，但如果想要拖動元素來滑動就不是這個屬性能幫上忙的地方了，還是要寫客製化 JS 去解決。

具體可以參考看看老牌投影片套件 [Swiper](https://swiperjs.com/demos#css-mode) 有個 CSS Mode 實際上就是用該屬性實作的。

更多實用範例與實作細節可以參考 [Practical CSS Scroll Snapping - CSS-Tricks](https://css-tricks.com/practical-css-scroll-snapping/)
