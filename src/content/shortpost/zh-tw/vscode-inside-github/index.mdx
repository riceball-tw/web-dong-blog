我沒打錯，試試看在任意 GitHub Repo 按下鍵盤上的「.」就可以打開一個完整的 VSCode 編輯器，雖然相較本地開發有些限制，不過還是挺震撼的吧！

比較有感的是在翻一些陌生的 Repo 不用下載就可以有完整的編輯體驗，或者是開發環境不在手邊隨便找台電腦打開瀏覽器也能立刻編輯專案。
