身為前端工程師，理論上工作環境都會配備多種瀏覽器並沒事就檢測看看做好的新功能是基礎操作。

但事實上很多時候自己常用的環境 OK 就默認沒問題推上去了，而一直切換瀏覽器和版本顯得拉高成本不實際，不知道有什麼方式解決這個問題？

我常用 Firefox 而其餘團隊成員則都用 Chrome ，意外很多 Firefox 上特有的 BUG 都是由我觀測到的，算是一種意外的多樣性 🙂。
