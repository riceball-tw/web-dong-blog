前端最有感的的快捷鍵天天按這兩顆按鈕絕對有上百次不為過吧！

- 檢視頁面元素：Ctrl + Shift + C
- 裝置尺寸預覽：Ctrl + Shift + M

一些瀏覽器會需要開 DevTools (F12) 才會生效
