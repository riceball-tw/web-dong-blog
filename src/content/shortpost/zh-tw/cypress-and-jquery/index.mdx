在用 Cypress 寫 End-to-End 測試時總是有一種回到家熟悉的感覺，查了一下原來背後其實一些方法是由 Chai-jQuery 提供的，沒想到一個已經普遍被視為過時的技術還在另一個領域發光發熱。

不知道有多少人前端開發啟蒙階段碰的不是 JavaScript 而是 jQuery，想到就很懷念！但拜託不要讓我回到以前渾沌的網路時代 🫠。

說到 jQuery 讓我想到有人拿它做了一個[線上作業系統](https://puter.com/)，酷！
