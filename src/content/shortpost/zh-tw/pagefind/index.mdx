所有使用靜態生成的網站都可以參考看看 pagefind 這個 NPM 套件。

pagefind 是一個以 Rust 編寫並且透過執行 WebAssembly 來實現的搜尋套件，它的誕生原因從一開始就非常明確：「替任何靜態網頁提供現成可用的搜尋方案」

- 搜尋快速
- 提供現成 UI
- 支援中日語系搜尋
- 支援任何靜態文件搜尋
- 支援分類、排序、自定義搜尋頁面 Meta 資料、索引重要程度調整

近期我的部落格也從 Fuse.js (模糊搜索套件) 整個換成 pagefind 囉，推薦任何寫靜態部落格的人試試看，pagefind 剛出正式 1.0 版本。
