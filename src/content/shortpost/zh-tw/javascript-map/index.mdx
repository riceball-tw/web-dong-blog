JavaScript ES6 中有一個用法與物件近似的資料結構我一直不是很清楚用途。

寫完文章總結：把 Map 當作是用來頻繁讀寫的物件，具備更好的性能、更明確的語法，而物件可以用作儲存固定的鍵值，在不需要頻繁讀寫的情況下使用。
