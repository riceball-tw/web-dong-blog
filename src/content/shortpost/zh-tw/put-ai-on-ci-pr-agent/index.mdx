最近都在處理 CI 設施，想說如果把每個 PR 都餵給 AI，自動化挑一些簡單的錯誤或文件補充，應該可以幫整個團隊節省時間並提升效率。

最終是挑了 Codium 的 PR Agent 加入 CI 當中，因為很常見且沒有太多供應商綁定的感覺，目前覺得好用的點：

- 安裝方便 (現成 CI 腳本，調整兩三下就能放上去運作)
- 允許多種 Git 服務供應商、多種架設方式、多種模型可自定義

也可以買他們的 PR-Agent Pro 不過不是必要的，只需要調整一下設定檔隨時都能改用自備的模型，如果對於把 AI 放上 CI 感興趣可以嘗試看看。

至於實際體驗？打算多用一陣子再來寫寫看心得。
