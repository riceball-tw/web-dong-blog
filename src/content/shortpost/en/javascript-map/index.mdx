In JavaScript ES6, there is a data structure similar to objects whose purpose I've never been quite clear about.

For more details, you can check the new article: [Understanding Built-in Data Structures in ES6 JavaScript: Map](https://www.webdong.dev/post/javascript-maps/)

The summary of the article is: treat Map as an object for frequent read and write operations, which offers better performance and clearer syntax, while objects can be used to store fixed key-value pairs when frequent read and write operations are not needed.
