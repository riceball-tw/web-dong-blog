一开始得知存在这个属性还蛮兴奋的，想象很多可以使用上的界面案例，甚至投影片功能也能轻松用原生卷轴来解决？不过实际用下去才会知道它们是截然不同的体验。

这项属性专注在卷轴的黏性吸附，因此无论是滚动卷轴还是触控时都能生效，但如果想要拖动元素来滑动就不是这个属性能帮上忙的地方了，还是要写客制化 JS 去解决。

具体可以参考看看老牌投影片套件 [Swiper](https://swiperjs.com/demos#css-mode) 有个 CSS Mode 实际上就是用该属性实现的。

更多实用范例与实现细节可以参考 [Practical CSS Scroll Snapping - CSS-Tricks](https://css-tricks.com/practical-css-scroll-snapping/)
