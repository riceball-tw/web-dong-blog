要在网页中使用任何种类的 Icon 强烈建议可以考虑使用 [Iconify](https://iconify.design/) (开源)。

![Webdong Posts](./webdong-posts.webp)

我自己公司原先是使用 FontAwesome，博客则是自己找 Icon 开 Figma 切好且用 SVGO 去压缩，不管哪种方式都超麻烦！还时常有找不到理想 Icon 的问题，直到项目架构升级无意间碰到 Nuxt icon 可以引入 Iconify 的数据集就爱上这个套件，现在想用哪款 Icon 就用，省了一堆麻烦问题。

今日博客更新推出也是将所有文章换成用 Iconify：

- Inline SVG 直接砍掉近百个图片请求
- 文章页 170 kb -> 93 kb 就算不马上用，到 [Icônes](https://icones.js.org/) 找找各式 Icon 也是挺方便的。
