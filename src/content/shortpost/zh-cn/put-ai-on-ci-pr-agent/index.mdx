最近都在处理 CI 设施，想说如果把每个 PR 都喂给 AI，自动化挑一些简单的错误或文件补充，应该可以帮整个团队节省时间并提升效率。

最终是挑了 Codium 的 PR Agent 加入 CI 当中，因为很常见且没有太多供应商绑定的感觉，目前觉得好用的点：

- 安装方便 (现成 CI 脚本，调整两三下就能放上去运作)
- 允许多种 Git 服务供应商、多种架设方式、多种模型可自定义

也可以买他们的 PR-Agent Pro 不过不是必要的，只需要调整一下设置档随时都能改用自备的模型，如果对于把 AI 放上 CI 感兴趣可以尝试看看。

至于实际体验？打算多用一阵子再来写写看心得。
