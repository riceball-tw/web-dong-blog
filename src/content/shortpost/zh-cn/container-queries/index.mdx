Container Queries 也已经有 90% 的全球支持度，将会在网页组件化的时代成为构建独立 RWD 组件样式的重要工具。

如今我们更倾向构建网页组件并灵活设置使其适应各种设备尺寸，不再像早期 AWD 强调针对特定设备尺寸提供特定的设计，这也是为什么已经有了 Media Queries 却还需要再推出 Container Queries 的原因——「观察特定元网页元素的变化并套用不同样式」

- Media Queries：易于指定整体页面样式
- Container Queries：易于指定个别元素在不同尺寸时样式

网页东东新推出[相关文章](https://www.webdong.dev/post/container-queries/)。

要详细了解 Container Queries 与其范例强烈推荐参考看看 [Ahmad Shadeed 的网站](https://ishadeed.com/article/css-container-query-guide/#introduction)，互动范例真的很厉害！
