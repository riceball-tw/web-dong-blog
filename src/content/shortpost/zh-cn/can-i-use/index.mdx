随着各项浏览器 API 的推动与统一，前端也越来越多工具可以用于构建网页，但要怎么知道 API 在不同的浏览器的支持程度、潜在问题与全球支持占比呢？可以试试看 [Can I Use](https://caniuse.com/) 网站，一项[由社区维护](https://github.com/Fyrd/caniuse) 的浏览器支持程度搜索网站。

![Can I Use 网站](./caniuse-overview.webp)

近期比较有感的是在制作一个标题清单，当点击清单项目时就让画面滑动到对应的标题，在更早以前会需要使用到 JS 的 `scrollTo()` 去计算好目标元素与当前位置需要滚动多少距离，但换一支更新颖专门用途的的 API `scrollIntoView()`就只要指定要滑动到的标题！省了许多功夫。

在使用一些新颖的 API 时我都会上这个网站确保目标客群的版本可以支持，团队也可以相互依照这个网站为基准相互监督。
