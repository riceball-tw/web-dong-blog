我没打错，试试在任意 GitHub Repo 按下键盘上的「.」就可以打开一个完整的 VSCode 编辑器，虽然相较本地开发有些限制，不过还是挺震撼的吧！

比较有感的是在翻一些陌生的 Repo 不用下载就可以有完整的编辑体验，或者是开发环境不在手边随便找台电脑打开浏览器也能立刻编辑专案。
