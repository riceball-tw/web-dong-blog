所有使用静态生成的网站都可以参考看看 pagefind 这个 NPM 套件。

pagefind 是一个以 Rust 编写并且通过执行 WebAssembly 来实现的搜索套件，它的诞生原因从一开始就非常明确：“替任何静态网页提供现成可用的搜索方案”

- 搜索快速
- 提供现成 UI
- 支持中日语系搜索
- 支持任何静态文件搜索
- 支持分类、排序、自定义搜索页面 Meta 资料、索引重要程度调整

近期我的博客也从 Fuse.js (模糊搜索套件) 整个换成 pagefind 啦，推荐任何写静态博客的人试试看，pagefind 刚出正式 1.0 版本。
