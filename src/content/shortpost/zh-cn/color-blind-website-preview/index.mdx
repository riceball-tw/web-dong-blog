刚好找到这个 [Toptal Colorblind Web Page Filter](https://www.toptal.com/designers/colorfilter) 可以即时预览不同线上网站在不同色盲滤镜的样貌。生理限制或环境都有可能影响我们接收信息的正确性，因此可以试着多留意用户适用性方面的需求，具体来说像是：WCAG、APCA。

- Protanopia = 红绿色盲（主要红色）
- Deuteranopia = 红绿色盲（主要绿色）
- Tritanopia = 蓝黄色盲
