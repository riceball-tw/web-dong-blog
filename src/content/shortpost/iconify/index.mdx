要在網頁中使用任何種類的 Icon 強烈建議可以考慮使用 [Iconify](https://iconify.design/) (開源)。

![Webdong Posts](./webdong-posts.webp)

我自己公司原先是使用 FontAwesome，部落格則是自己找 Icon 開 Figma 切好且用 SVGO 去壓縮，不管哪種方式都超麻煩！還時常有找不到理想 Icon 的問題，直到專案架構升級無意間碰到 Nuxt icon 可以引入 Iconify 的資料集就愛上這個套件，現在想用哪款 Icon 就用，省了一堆麻煩問題。

今日部落格更新推出也是將所有文章換成用 Iconify：

- Inline SVG 直接砍掉近百個圖片請求
- 文章頁 170 kb -> 93 kb 就算不馬上用，到 [Icônes](https://icones.js.org/) 找找各式 Icon 也是挺方便的。
